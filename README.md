# Poke-app

This project is an app to search pokemon by name or id. The project live version is available on <https://poke-app-teal-eight.vercel.app>.

## How to run the project locally?

1. Install Node.js
2. Clone repository
3. Run `npm install`
4. Run `npm run dev`
5. Open <http://localhost:3000>
